# 🐍 贪吃蛇

一个用纯 HTML/CSS/JavaScript 实现的经典贪吃蛇网页小游戏，暗色霓虹风格，支持键盘和触屏操作。

## 🎮 玩法

- 使用 **方向键** 或 **WASD** 控制蛇的移动方向
- 吃到食物得分，蛇身变长，速度逐渐加快
- 撞墙或撞到自己则游戏结束
- 按 **空格键** 暂停 / 继续
- 按 **R 键** 重新开始

## 🍎 食物

| 类型 | 外观 | 分值 | 出现概率 |
|------|------|------|----------|
| 普通苹果 | 🔴 红色 | 1 分 | 85% |
| 金色星星 | ⭐ 金光 | 5 分 | 15% |

## ✨ 特性

- 🎨 暗色主题 + 霓虹光效，蛇身渐变色
- 📱 支持移动端触屏滑动操作
- 🏆 最高分自动保存在浏览器本地存储
- ⚡ 每吃 5 个食物速度提升一档
- 🖼️ 游戏结束弹窗动画，一键重开

## 🚀 快速开始

直接用浏览器打开 `index.html` 即可游玩：

```bash
open index.html
```

或者部署到任意静态文件服务器。
